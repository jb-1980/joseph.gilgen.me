# TIC TAC TOE

This was a fun project for me. It was the first game in [freeCodeCamp's](https://www.freecodecamp.org) curriculum, and really the first game I had built. I suppose I had fancied building video games when I was a kid, so this was a small taste of that.

The AI is _super_ basic. It just basically looks for an empty space and plays there. Which makes it _super_ easy to win. But hey, my 4-year old got a kick out of it.
