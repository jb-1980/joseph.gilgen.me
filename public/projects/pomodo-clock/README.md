# Pomodo Clock

This is another [FreeCodeCamp](https://www.freecodecamp.org) project. I really had a lot of fun with this one. I was challenged to use the canvas object, and I really felt my intellect stimulated as I processed the challenge. In the end, I thought it came out really nice. I think it is really cool how the clock fills the circle with green for work time, and then empties with red for the break. It was such a feeling of accomplishment when I was able to see it fill and empty at different rates, based on the amount of time alloted.
